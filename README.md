# PromTech Marketplace

PromTech Marketplace — frontend pet-project маркетплейса промышленного оборудования на `React + TypeScript + Vite`.
Проект демонстрирует витрину каталога, поиск и фильтрацию товаров, сценарии личного кабинета продавца и архитектурный подход к построению B2B-интерфейса.

Репозиторий подготовлен как презентационный проект для GitHub-портфолио и находится в активной разработке. Текущая версия фокусируется на качестве интерфейса, структуре приложения и ключевых пользовательских сценариях.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. При необходимости укажите `VITE_API_BASE_URL` в `.env.local`
3. Run the app:
   `npm run dev`

## Обзор проекта

Идея проекта — показать, как может выглядеть современный маркетплейс промышленного оборудования с точки зрения покупателя и продавца.

Сейчас в проекте реализованы:
- каталог товаров с карточками и детальным просмотром
- навигация по категориям и подкатегориям
- поиск и клиентская фильтрация товаров
- личный кабинет продавца
- базовое управление товарами
- группировка товарных позиций
- моковые данные для визуализации интерфейса и демонстрации UX

## Основные возможности

- витрина каталога промышленного оборудования
- карточки товаров с характеристиками и быстрыми действиями
- модальное окно товара с галереей и подробной информацией
- поиск по товарам
- фильтрация по категориям и подкатегориям
- личный кабинет продавца
- добавление, редактирование и удаление товарных позиций
- интерфейс управления группами товаров
- адаптивная основа для дальнейшего развития проекта
